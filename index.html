What is it?

ioBrew is a webapp I created for keeping track of drinks owed between friends in a fun, social way Some of the features are cross user functionality, achievement tracking, and automatic email reminders, just to name a few. For this project I built out both the frontend and the backend, along with creating the web page's design.
